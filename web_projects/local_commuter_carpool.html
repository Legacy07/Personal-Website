Local Commuter Carpool

Overview

This is a website for local residents to find commuter carpool. The website allows for;

  • Casual users to Search for commutes only.
  • Registered users to Post Journey and Edit/Delete posts.
  • Registered users to View detailed search results.



Technologies

  • XHTML
  • PHP
  • CSS
  • MySQL
